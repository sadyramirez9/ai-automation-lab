# AI Automation Lab

This repository is a hands-on lab for learning how to apply AI to automate knowledge work and improve personal productivity.

## Purpose
- Build practical AI-powered automations
- Develop reusable prompt systems
- Experiment with workflows that reduce manual work

## Repository Structure
- `prompts/` — Prompt templates and prompt systems
- `scripts/` — Python scripts and automation utilities
- `workflows/` — End-to-end automation flows
- `notes/` — Learning notes and experiments
- `capstone/` — Larger, integrated automation projects

## How I Use This Repo
This repo evolves incrementally as I test ideas, build tools, and refine AI-driven workflows in real-world scenarios.
